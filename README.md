# Локальный Live Share для Python

Небольшое приложение для синхронизации файла `liveshare.py` между двумя компьютерами в одной локальной сети.

## Требования

- Python 3.8+
- `pip install -r requirements.txt`

## Состав

- `sync_server.py` - WebSocket-сервер на машине учителя
- `sync_client.py` - клиент синхронизации на обеих машинах
- `liveshare.py` - общий рабочий Python-файл

## Запуск

На компьютере учителя:

```bash
python sync_server.py
python sync_client.py
```

На компьютере ученика:

```bash
python sync_client.py 192.168.1.X
```

Вместо `192.168.1.X` укажите локальный IP компьютера учителя.

## Как это работает

1. Оба открывают `liveshare.py` в VS Code или любом другом редакторе.
2. Любое изменение файла отправляется на сервер примерно каждые 300 мс.
3. Второй компьютер получает новую версию и записывает её в свой `liveshare.py`.
4. Каждый запускает `python liveshare.py` локально у себя.

## Ограничения

- Конфликты не разрешаются, используется last write wins.
- Синхронизируется только `liveshare.py`.
- Интернет не нужен после установки зависимости.
